TEC News Portal

A modern, full-stack news portal system built for Telecommunication Engineering College in Malaysia. Provides complete content management with writer dashboards, admin controls, and a public-facing news portal.

Cloudflare Workers Cloudflare D1 Cloudflare R2 JWT Authentication Multi-language Serverless

Overview

TEC News Portal is a modern, full-stack news portal system built entirely on Cloudflare's edge network. It provides a complete content management solution with writer dashboards, admin controls, and a public-facing news portal. The system features multi-language support (English and Malay), responsive design with dark/light theme, integrated media management, and SEO optimization with dynamic meta tags for social media sharing.

Architecture Overview

The system follows a serverless architecture with clear separation between frontend (Cloudflare Pages) and backend (Cloudflare Workers). The architecture includes multiple specialized workers for different functionalities, ensuring scalability and maintainability.

API Endpoints

Authentication

  • POST /api/auth/login - Login user and receive JWT token
  • POST /api/auth/validate - Validate JWT token
  • GET /api/profile - Get user profile
  • PUT /api/profile - Update user profile

Articles

  • GET /api/articles/published - Get published articles with pagination
  • GET /api/articles/featured-public - Get featured articles for homepage
  • GET /api/articles/search?q={query} - Search published articles
  • GET /api/articles/{id} - Get single article by ID
  • POST /api/articles/submit - Submit new article for review

Admin

  • GET /api/admin/stats - Get dashboard statistics
  • GET /api/admin/articles/pending - Get pending articles for review
  • POST /api/admin/articles/{id}/approve - Approve article for publication
  • POST /api/admin/articles/{id}/reject - Reject article
  • GET /api/admin/users - List all users

Upload

  • POST /api/upload/image - Upload image file to R2

Challenges & Solutions

Developing TEC News Portal involved several technical challenges to ensure a fast, secure, and scalable application.

Challenge: Serverless Architecture Complexity

Managing multiple Cloudflare Workers, coordinating between them, and ensuring proper data flow across the serverless infrastructure.

Solution

  • Created specialized workers for different functionalities (auth, articles, admin, upload, preview)
  • Implemented consistent API patterns across all workers
  • Used shared environment variables and configurations
  • Created deployment scripts for streamlined worker deployment

Result: Maintainable and scalable serverless architecture.

Challenge: Dynamic Meta Tags for Social Sharing

Ensuring proper social media previews (Open Graph, Twitter Cards) for shared articles without server-side rendering.

Solution

  • Created Preview Worker using HTMLRewriter API
  • Intercepted article detail page requests
  • Fetched article data from API and dynamically rewrote meta tags
  • Optimized for Facebook, Twitter, LinkedIn sharing

Result: Perfect social media previews with dynamic content.

Challenge: Multi-language Support

Implementing bilingual support (English/Malay) across all pages and maintaining translation consistency.

Solution

  • Created JSON-based translation files (en.json, ms.json)
  • Built LanguageManager class for dynamic language switching
  • Persisted language preference in localStorage
  • Applied translations to all UI elements and content

Result: Seamless bilingual experience for users.
